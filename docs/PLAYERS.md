## 角色系统概述

### 设计理念

每个角色都有独特的技能机制和玩法风格，从几何控制到火焰法术，从陷阱布置到雷电风暴，提供多样化的游戏体验。

### 共同特性

所有角色共享以下基础机制：

- **移动**: WASD键控制移动
- **冲刺**: 鼠标左键向鼠标方向冲刺并造成伤害
- **Q技能**: 按住蓄力，松开释放（大部分角色进入子弹时间）
- **E技能**: 立即释放的技能
- **能量系统**: 技能消耗能量，能量自动恢复
- **护甲系统**: 最多3点护甲，减少受到的伤害

### 角色切换

- 按 **Tab键** 可在7个角色之间循环切换（调试功能）
- 切换后保留当前波次和升级

## 角色列表

### 1. Herder（牧者）

<div align="center">

**几何控制专家 | 区域控制 + 高爆发**

</div>

#### 角色定位

牧者是一个需要规划和预判的高风险高回报角色。通过规划冲刺路径形成闭环，可以瞬间消灭圈内所有敌人。

#### 基础属性

| 属性 | 数值 |
|------|------|
| 生命值 | 5000 |
| 移动速度 | 500 |
| 冲刺距离 | 340 |
| 冲刺速度 | 1700 |
| 冲刺伤害 | 14 |
| 最大能量 | 999 |
| 初始能量 | 500 |

#### 技能详解

##### 左键 - 冲刺攻击
- **类型**: 基础攻击
- **消耗**: 10能量
- **效果**: 向鼠标方向冲刺，对路径上的敌人造成伤害

##### Q技能 - 规划路径
- **类型**: 规划技能
- **消耗**: 20能量/段
- **操作**:
  1. 按住Q键进入子弹时间（0.1倍速）
  2. 左键点击规划冲刺路径（可规划多段）
  3. 松开Q键执行路径
- **效果**:
  - 沿规划路径冲刺
  - 如果路径形成闭环，触发"闭环绞杀"
  - **闭环绞杀**: 瞬杀圈内所有敌人（刺猬免疫）
- **视觉**: 规划时显示红色几何线条

##### E技能 - 范围爆炸
- **类型**: 范围伤害
- **消耗**: 20能量
- **效果**:
  - 以自身为中心的大范围爆炸（200像素半径）
  - 造成伤害并击退敌人
  - 成功击杀敌人后恢复1点护甲
- **冷却**: 无冷却

#### 玩法技巧

1. **规划路径**: 在敌人密集区域规划闭环，一次性清场
2. **能量管理**: 规划路径消耗大量能量，注意能量余量
3. **护甲恢复**: 使用E技能击杀敌人恢复护甲
4. **风险控制**: 规划时进入子弹时间，但执行时无保护

#### 适合玩家

- 喜欢策略和规划的玩家
- 追求高风险高回报的玩家
- 擅长预判敌人位置的玩家

### 2. Butcher（屠夫）

<div align="center">

**电锯狂魔 | 持续伤害 + 区域压制**

</div>

#### 角色定位

屠夫通过冲刺留下血色裂痕，创造持续伤害区域。可以创造"死亡角斗场"，在区域内获得移速加成。

#### 基础属性

| 属性 | 数值 |
|------|------|
| 生命值 | 5000 |
| 移动速度 | 500 |
| 冲刺距离 | 400 |
| 冲刺速度 | 2000 |
| 冲刺伤害 | 20 |
| 最大能量 | 999 |
| 初始能量 | 800 |

#### 技能详解

##### 左键 - 电锯冲刺
- **类型**: 基础攻击
- **消耗**: 10能量
- **效果**:
  - 冲刺并造成伤害
  - 冲刺路径留下血色裂痕（持续5秒）
  - 裂痕对触碰的敌人造成持续伤害
  - 敌人死亡时生成血肉地雷

##### Q技能 - 死亡角斗场
- **类型**: 区域控制
- **消耗**: 50能量
- **操作**: 按住Q规划，松开Q在鼠标位置创造圆形区域
- **效果**:
  - 创造大范围圆形区域（持续8秒）
  - 区域内敌人持续受伤
  - 玩家在区域内移速提升60%
  - 敌人死亡生成血肉地雷
- **视觉**: 红色圆形区域

##### E技能 - 裂痕回拉
- **类型**: 控制技能
- **消耗**: 30能量
- **效果**:
  - 将所有裂痕拉回自身
  - 路径上的敌人被拉扯并受伤
  - 清除所有裂痕
- **用途**: 聚怪、清理裂痕

#### 玩法技巧

1. **裂痕布置**: 在战场上布置裂痕网络，创造危险区域
2. **角斗场**: 在敌人密集区域创造角斗场，利用移速加成风筝敌人
3. **血肉地雷**: 敌人死亡生成地雷，可以连锁爆炸
4. **裂痕回拉**: 聚集敌人后使用E技能拉回，配合角斗场清场

#### 适合玩家

- 喜欢持续伤害的玩家
- 擅长区域控制的玩家
- 喜欢创造战场优势的玩家

### 3. Weaver（织网者）

<div align="center">

**蛛网陷阱大师 | 控制 + 陷阱布置**

</div>

#### 角色定位

织网者通过标记敌人为锚点，连线形成蛛网。可以瞬杀普通敌人，对刺猬造成双倍伤害。

#### 基础属性

| 属性 | 数值 |
|------|------|
| 生命值 | 5000 |
| 移动速度 | 500 |
| 冲刺距离 | 320 |
| 冲刺速度 | 1600 |
| 冲刺伤害 | 12 |
| 最大能量 | 999 |
| 初始能量 | 600 |

#### 技能详解

##### 左键 - 吐丝冲刺
- **类型**: 基础攻击
- **消耗**: 10能量
- **效果**:
  - 冲刺并造成伤害
  - 冲刺路径上的敌人被标记为锚点
  - 锚点之间自动连线（白色线条）
- **视觉**: 敌人头顶显示锚点标记

##### Q技能 - 收网
- **类型**: 爆发伤害
- **消耗**: 20能量
- **效果**:
  - 激活所有锚点之间的连线
  - 普通敌人瞬杀
  - 刺猬受到双倍伤害（不会瞬杀）
  - 连线切割路径上的敌人
  - **特殊**: 玩家在网内且有硬壳龟锚点时恢复护甲
- **视觉**: 连线变为红色并闪烁

##### E技能 - 蜘蛛地雷
- **类型**: 陷阱
- **消耗**: 30能量
- **效果**:
  - 在脚下放置陷阱（持续10秒）
  - 敌人触发后范围爆炸
  - 爆炸范围内所有敌人被禁锢3秒并标记为锚点
  - 造成伤害
- **用途**: 控制敌人移动，配合Q技能

#### 玩法技巧

1. **锚点标记**: 冲刺标记多个敌人为锚点
2. **收网时机**: 等待敌人聚集后使用Q技能收网
3. **地雷布置**: 提前布置地雷，引导敌人触发
4. **护甲恢复**: 标记硬壳龟后在网内收网可恢复护甲

#### 适合玩家

- 喜欢控制和陷阱的玩家
- 擅长时机把握的玩家
- 喜欢一击必杀的玩家

## 技能系统

### 技能类型

#### 1. 规划技能（Q技能）

- **特点**: 按住Q进入子弹时间，规划路径后松开执行
- **角色**: Herder、Pyro、Wind、Sapper
- **优势**: 可以精确规划，形成闭环触发特殊效果
- **劣势**: 需要时间规划，执行时无保护

#### 2. 即时技能（E技能）

- **特点**: 按E立即释放
- **角色**: 所有角色
- **优势**: 快速释放，适合紧急情况
- **劣势**: 无法规划，需要时机把握

#### 3. 区域技能

- **特点**: 创造持续伤害或控制区域
- **角色**: Butcher、Pyro、Tempest
- **优势**: 持续效果，无需持续操作
- **劣势**: 固定位置，敌人可以躲避

#### 4. 陷阱技能

- **特点**: 提前布置，敌人触发
- **角色**: Weaver、Sapper
- **优势**: 提前布置，控制战场
- **劣势**: 需要预判，敌人可能不触发

### 能量管理

- **能量消耗**: 所有技能都消耗能量
- **能量恢复**: 每秒自动恢复0.5能量
- **击杀回能**: 可通过升级获得击杀回能属性
- **能量上限**: 可通过升级提升能量上限

#### Pyro（烈焰）

PolyLash的角色系统设计注重：

1. **多样性**: 7个角色，7种玩法

3. **平衡**: 所有角色都有优势和劣势
4. **可扩展**: 易于添加新角色
